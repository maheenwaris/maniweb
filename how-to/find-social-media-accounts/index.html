Find Social Media Accounts with Sherlock

January 11, 2023

  • Penetration Testing

Brief

Hunt down social media accounts by username across social networks with Sherlock

This tool will only work for people who use the same username across all social media platforms.

Installation

git clone https://github.com/sherlock-project/sherlock
cd sherlock
sudo pip3 install -r requirements.txt

Find Social Accounts

python3 sherlock [username]

For example:

python3 sherlock mansoor.yuh

you can see other options with -h

usage: sherlock [-h] [--version] [--verbose] [--folderoutput FOLDEROUTPUT]
                [--output OUTPUT] [--tor] [--unique-tor] [--csv]
                [--site SITE_NAME] [--proxy PROXY_URL] [--json JSON_FILE]
                [--timeout TIMEOUT] [--print-all] [--print-found] [--no-color]
                [--browse] [--local]
                USERNAMES [USERNAMES ...]

Recommended Command

python3 sherlock [username] --output [filename]

For example:

python3 sherlock mansoor.yuh --output sherlock.txt

--output the command output is saved to a file

that’s it <3
