Hugo Server on LAN

August 18, 2022

  • hugo
  • server
  • LAN

Brief

[IP-ADD] is your local IP address.

Command

use this command instead of “hugo server”

hugo server --bind [IP-ADD] --baseURL http://[IP-ADD]

Conclusion

visit http://[IP-ADD] from any device on your LAN to view the temporary hugo site.
